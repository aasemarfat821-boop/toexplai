JavaScript Loops

:يعني إيه Loops

Loops هي الحلقات اللي بنكرر فيها كود معين أكتر من مرة بدل ما نكتب الكود أكتر من مرة.

:أشهر أنواع الحلقات

  • for loop
  • while loop
  • do...while loop

:مثال على for loop

for (let i = 1; i <= 5; i++) {   console.log(i); }

1️⃣ الحلقة for

:مثال على while loop

let i = 1; while(i <= 5) {   console.log(i);   i++; }

2️⃣ الحلقة while
