Integrating CSS & JavaScript

ما هو CSS؟

CSS هو اختصار لـ "Cascading Style Sheets".
يستخدم لتنسيق شكل وتصميم الصفحة مثل الألوان والخطوط والخلفيات.

1️⃣ ربط CSS مع HTML

ما هو JavaScript؟

JavaScript لغة برمجة تُستخدم لإضافة التفاعلية للموقع.
يمكن من خلالها التحكم في المحتوى والرسوميات والصفحات بشكل ديناميكي.

2️⃣ ربط JavaScript مع HTML

كيفية دمج CSS و JavaScript

يمكن ربط CSS و JavaScript بالصفحة مباشرة داخل <style> و <script> أو عن طريق ملفات خارجية.
هذا يسمح بتصميم الصفحة بشكل جميل وإضافة تفاعلات للمستخدم.
