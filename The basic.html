The Basic Rule of CSS

ما هو القاعدة الأساسية في CSS؟

القاعدة الأساسية في CSS تتكون من جزأين رئيسيين:
Selector (المحدد): العنصر الذي تريد تغييره.
Declaration (التعليمات): مجموعة الخصائص والقيم التي تطبق على العنصر.

مثال على القاعدة الأساسية

إذا أردت تغيير لون النص داخل الفقرة إلى الأزرق:
p { color: blue; }
هنا p هو الـ Selector، و color: blue; هو Declaration.

مكونات Declaration

Declaration تتكون من:
1. property (الخاصية) مثل color أو font-size.
2. value (القيمة) مثل blue أو 16px.
3. كل Declaration تنتهي بفاصلة منقوطة ;
