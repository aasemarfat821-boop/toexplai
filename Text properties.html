Text Properties in CSS

ما هي Text Properties؟

Text Properties هي الخصائص التي تتحكم في مظهر النصوص داخل الصفحة.
يمكن التحكم في اللون، الحجم، نوع الخط، المحاذاة، والعديد من الخصائص الأخرى.

أمثلة على Text Properties

1. color: لتغيير لون النص، مثال color: blue;
2. font-size: لتغيير حجم الخط، مثال font-size: 16px;
3. font-family: لتحديد نوع الخط، مثال font-family: Arial;
4. text-align: لتحديد محاذاة النص، مثال text-align: center;
5. font-weight: لتحديد سماكة الخط، مثال font-weight: bold.

مثال عملي

p { color: red; font-size: 18px; text-align: center; font-weight: bold; }
