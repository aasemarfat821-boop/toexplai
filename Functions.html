JavaScript Functions

:يعني إيه Functions

Functions هي دوال أو كتل كود بنسميها باسم و نقدر نستدعيها وقت ما نحتاج لتنفيذ نفس الكود.

:فوائد Functions

- إعادة استخدام الكود بدل ما نكتبه أكتر من مرة - تنظيم الكود ووضوحه - سهولة التعديل والتحكم

1️⃣ إنشاء دالة بسيطة

:مثال على Function بسيطة

function sayHello() {   console.log("Hello World!"); } sayHello();

:Function مع مدخلات (Parameters)

function greet(name) {   console.log("Hello " + name); } greet("Ali");
