Change the Shape of Buttons

كيفية تغيير شكل الأزرار؟

يمكن تغيير شكل الأزرار في CSS باستخدام خاصية border-radius.
هذه الخاصية تتحكم في انحناء زوايا الزرار.

أمثلة على تغيير شكل الأزرار

1. زر عادي: <button>Click me</button>
2. زر بزوايا مستديرة: button { border-radius: 10px; }
3. زر بشكل دائرة: button { border-radius: 50%; width: 100px; height: 100px; }
