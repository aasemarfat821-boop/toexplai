Save the File As

ما معنى "Save the file as"؟

"Save the file as" يعني حفظ الملف باسم جديد أو في مكان معين على جهازك.
يستخدم هذا الخيار لتحديد اسم الملف وامتداده والمكان الذي تريد حفظه فيه.

خطوات استخدام "Save the file as"

1. افتح الملف الذي تريد حفظه.
2. اختر File من القائمة العليا.
3. اضغط على "Save the file as".
4. اختر المكان والاسم والامتداد المناسب للملف.
5. اضغط Save.

أهمية "Save the file as"

يتيح لك هذا الخيار إنشاء نسخ متعددة من الملف بنفس المحتوى.
كما يسمح بتغيير الامتداد لتوافق أنواع الملفات المختلفة.
