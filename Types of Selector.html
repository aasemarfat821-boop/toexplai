Types of Selectors in CSS

ما هي Selectors؟

Selector هو الجزء من القاعدة في CSS الذي يحدد العنصر الذي سيتم تطبيق التنسيقات عليه.
يستخدم لتحديد العناصر المختلفة مثل الفقرات، العناوين، الصور، أو أي عنصر HTML آخر.

أنواع Selectors الأساسية

1. Type Selector: يحدد العناصر بالاسم، مثل p أو h1.
2. Class Selector: يحدد العناصر باستخدام class، مثل .section-text.
3. ID Selector: يحدد عنصر واحد باستخدام id، مثل #title.
4. Universal Selector: يحدد كل العناصر باستخدام *.
5. Grouping Selector: لتطبيق نفس التنسيق على عدة عناصر، مثل h1, p, div.

مثال عملي على Type و Class Selector

p { color: blue; } → Type Selector يغير لون جميع الفقرات.
.section-text { font-size: 16px; } → Class Selector يغير حجم الخط للعناصر التي تحمل هذا class.
